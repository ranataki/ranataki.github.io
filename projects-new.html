CESSNA LANDING GEAR FEA

A finite element analysis project for ME 123, investigating the structural behavior of Cessna aircraft landing gear under various loading conditions. The study examined different geometric configurations and material thicknesses to optimize performance.

Using ParaView for visualization and FEniCS for computational analysis, I developed detailed models that accurately predicted stress distributions, deformation patterns, and failure modes under realistic operating conditions.

The analysis included parametric studies varying landing gear geometry, material properties, and loading scenarios to identify optimal design configurations for weight reduction while maintaining structural integrity.

Analysis Capabilities:

  • 3D finite element mesh generation and refinement
  • Multi-physics simulation including structural and thermal effects
  • Parametric studies of geometric and material variables
  • Stress concentration analysis at critical joints
  • Fatigue life prediction under cyclic loading
  • Optimization algorithms for weight and performance
  • Comprehensive reporting and visualization of results
FEA ParaView FEniCS Structural Analysis Aerospace Engineering

IZMO CIGARETTE COLLECTION ROBOT

Izmo is an autonomous environmental cleanup robot designed to identify and collect cigarette butts from public spaces. The project combines computer vision, embedded systems, and mechanical design to create an effective solution for urban waste management.

The robot uses OpenCV-powered computer vision running on a Raspberry Pi to detect and locate cigarette butts in real-time. An Arduino microcontroller manages the mechanical systems including navigation, collection mechanisms, and sensor integration.

The project addresses a significant environmental challenge, as cigarette butts are among the most common forms of litter worldwide. Izmo demonstrates how robotics and AI can be applied to create practical solutions for environmental sustainability.

CS248B: CHARACTER ANIMATION PROJECTS

Throughout CS248B: Fundamentals of Character Animation, I developed three sophisticated gaming projects that deepened my understanding of advanced physics simulation and computer graphics. Each project built upon the previous one, creating a comprehensive learning experience in interactive media development.

Starting with a blob dynamics simulator, I explored the fundamentals of soft-body physics and fluid dynamics. The project required implementing complex mathematical models to simulate realistic blob behavior, including surface tension, viscosity, and collision responses.

The second project involved creating a physics-based pinball game, where I implemented rigid body dynamics, collision detection systems, and realistic ball physics. This project challenged me to optimize performance while maintaining accurate physical simulations.

The final project culminated in a character animation system that brought together all previous learnings. I implemented character kinematics using Jacobian matrices, advanced physics simulations with ray casting, penalty forces, and compression mechanics.

Key Technical Achievements:

  • Implemented Signed Distance Functions (SDFs) for efficient collision detection
  • Developed large-scale object interaction systems
  • Created character kinematics using Jacobian matrices
  • Built advanced physics simulations with ray casting
  • Implemented penalty forces and compression mechanics
  • Explored space-time motion algorithms
  • Optimized performance for real-time interactive applications

POMELO TEACHING ROBOT

Pomelo is an innovative teaching assistant robot designed to make learning computer science fundamentals fun and tangible for elementary school students. The project bridges the gap between abstract programming concepts and hands-on learning through interactive robotics.

The system uses Aruco markers (special visual codes) to read and interpret physical block arrangements created by students. Children arrange blocks to create pathways through mazes, and Pomelo brings their solutions to life by following their block-based instructions in real-time.

This approach transforms traditional computer science education by making algorithmic thinking physical and immediately visible. Students can see the direct results of their logical reasoning as Pomelo navigates through challenges they've designed.

Key Features:

  • Computer vision system using OpenCV for block recognition
  • Aruco marker detection and interpretation
  • Real-time path planning and navigation algorithms
  • Interactive maze-solving capabilities
  • Educational curriculum integration
  • Child-friendly robot design and safety features
  • Modular block system for various programming concepts
